# .bazel/go

A template for Bazel with Go.

## Variables

| Name             | Description       |
| ---------------- | ----------------- |
| `<<PROJECT>>`    | The project name  |
| `<<OWNER>>`      | The owner name    |
| `<<BAZEL_VERSION>>` | The version of Bazel |
| `<<GO_VERSION>>` | The version of Go |
